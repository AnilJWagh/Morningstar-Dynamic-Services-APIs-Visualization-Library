## Description

The Risk & Rating component displays a security's Morningstar Risk and Return rating, SRRI measurement, and other risk measures including Alpha, Beta, Sharpe Ration, and Standard Deviation.

## Fetching Data

The component is populated with data returned by calls to the <a href={SECURITY_DETAILS_URL} target="_blank">Security Details API</a>.
<!--
- [API Data Access Library](#api-data-access-library)
- [Request Example](#request)
- [Request Payload Example](#request-payload)
-->
### API Data Access Library

The <a href={DATA_ACCESS_LIBRARY} target="_blank">API Data Access Library</a> is used to fetch data from the API.

``` javascript dark
window.mstarApisSdk.securityDetails.getSecurityDetails(securityId);
```
<details>
  <summary><span class="summary-text">Example</span></summary>

``` javascript dark
const securityId = "F0GBR050DD";
window.mstarApisSdk.securityDetails.getSecurityDetails(securityId);
```
</details>

### Request Example

```javascript dark
https://www.emea-api.morningstar.com/ecint/v1/securities/{{securityId}}?viewId={{viewId}}&idtype={{idType}}&languageId={{languageId}}&currencyId={{currencyId}}&responseViewFormat={{responseViewFormat}}

```

### Request Payload Example

| Query Parameter | Description|Example |
| :----------- | :--------| :----------- |
| `id` | Unique identifier of a security, such as a MSID or ISIN.| `securityId=F0GBR050DD` |
| `viewId` |Unique identifier of a view.| `viewId=snapshot` |
| `idtype` | Type of security identifier used. |`idtype=msid` |
|` languageId`| 	ISO culture codes.| `languageId=en-GB`|
| `currencyId` | 	ISO alpha-3 currency code.| `currencyId=GBP`|
| `responseViewFormat` | Format of the output data. | `responseViewFormat=json` |

parsedModelData() {
  const tableData = [];
  const riskStatisticsData = this.modelData || [];
  const riskStatistics = riskStatisticsData.find(
    (data) => data.Type === 'MonthEnd' && data.CurrencyOption === 'DISPLAYCURRENCY',
  ) || {};
  const frequency = 'M';
  this.fieldDefinitions.forEach((fieldDef) => {
    const rowData = { dataPoint: fieldDef.text };
    this.periods.slice(1).forEach((timePeriod) => {
      const fieldData = riskStatistics[fieldDef.value] || [];
      const filteredObject = fieldData.find(
        (dataItem) => dataItem.Frequency === frequency
          && dataItem.TimePeriod === timePeriod.value,
      ) || {};
      rowData[timePeriod.value] = this.formatNumber(filteredObject.Value, 2) || '–';
    });
    tableData.push(rowData);
  });
